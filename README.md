# Project Description

The Contact Management System is a Python application that helps users store and manage contact information. Users can add, search, update, delete, and display contacts. The program also saves contacts in a JSON file, exports them to a CSV file, and displays contact statistics.

This project demonstrates the use of Python functions, dictionaries, loops, conditionals, file handling, regular expressions, and input validation.


# Project Objectives

* Learn how to use dictionaries to store data.
* Perform CRUD (Create, Read, Update, Delete) operations.
* Validate phone numbers and email addresses.
* Save and load data using JSON.
* Export data to CSV.
* Build a menu-driven Python application.
* Practice modular programming using functions.
